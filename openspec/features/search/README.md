# Search Feature

Reserved for future search-related feature specifications.

This folder follows the same feature-layer contract as the other business-facing OpenSpec areas.

## Product Content and Variants V1 Contract

Search remains a reserved future capability. When implemented, it shall consume
the shared public Product contract and may use approved Product identity,
descriptions and merchandising metadata without exposing SKU, barcode, reference
or deferred operational data.

Variant Attributes and tags do not become public filters or destinations in V1.
Any faceted search or attribute ranking requires a separate approved Search
specification. Search must preserve Product Type as the authoritative taxonomy
leaf and Product-level Commercial Availability price protection.
